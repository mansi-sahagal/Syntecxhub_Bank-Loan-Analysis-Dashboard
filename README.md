📊 Bank Loan Analysis Dashboard
📈 Project Title / Headline

Bank Loan Analysis Dashboard – Loan Performance & Portfolio Insights

An interactive Power BI dashboard designed to analyze loan applications, funded amounts, repayment performance, interest rates, loan quality, and customer borrowing patterns through dynamic visualizations and interactive filtering.

📌 Short Description / Purpose

The Bank Loan Analysis Dashboard is a comprehensive and visually engaging Power BI report developed to help financial institutions monitor loan performance, evaluate lending activities, and gain valuable insights into customer borrowing behavior. The dashboard provides key metrics such as Total Loan Applications, Total Funded Amount, Total Amount Received, Average Interest Rate, and Average Debt-to-Income (DTI) Ratio.

It also classifies loans into Good Loans and Bad Loans, enabling users to assess portfolio quality and repayment performance. Interactive visualizations provide insights into monthly loan application trends, loan purpose, home ownership, loan term, loan status, funded amount versus repayments, interest rates, and DTI ratios.

With interactive filters for different loan purposes, users can easily explore lending trends and identify risk factors. This dashboard supports data-driven decision-making for improving loan approval strategies, managing credit risk, and optimizing overall loan portfolio performance.

🛠 Tech Stack

The dashboard was built using the following tools and technologies:

📊 Power BI Desktop – Dashboard development and interactive visualizations
📂 Microsoft Excel / CSV Dataset – Bank loan dataset
🧮 DAX (Data Analysis Expressions) – Calculated measures, KPIs, and business metrics
🔄 Power Query Editor – Data cleaning, transformation, and preprocessing
📈 Data Modeling – Established relationships between multiple tables for efficient reporting
✨ Key Features
📋 Total Loan Applications KPI
💰 Total Funded Amount & Total Amount Received
📈 Average Interest Rate Analysis
📊 Average Debt-to-Income (DTI) Ratio
✅ Good Loan vs Bad Loan Performance Analysis
💵 Funded Amount vs Repayment Comparison
📅 Monthly Loan Application Trend
🎯 Loan Status Analysis (Fully Paid, Charged Off, Current)
🏠 Home Ownership-wise Loan Distribution
💳 Loan Purpose Analysis using Treemap
⏳ Loan Term Analysis (36 Months vs 60 Months)
🎛 Interactive Loan Purpose Filter Panel
🎨 Modern, responsive, and user-friendly dashboard design
📊 Business Impact & Insights

This dashboard helps banks and financial institutions to:

Monitor overall loan portfolio performance.
Evaluate loan funding and repayment efficiency.
Identify the percentage of Good Loans and Bad Loans.
Analyze customer borrowing trends over time.
Compare loan performance across different loan purposes.
Understand customer profiles based on home ownership and loan terms.
Monitor interest rates and DTI ratios for better risk assessment.
Support data-driven lending and credit risk management decisions.
